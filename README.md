# Dashboard-for-UPPSOTP
A dashboard that stores uploaded information in editable blocks.

## ⚙️ Требования

Убедитесь, что у вас установлены:

- **Node.js** `>= 16.x`
  [Скачать с официального сайта](https://nodejs.org/)

- **npm** (устанавливается вместе с Node.js)

- **Angular CLI** `>= 16.x`
  Установите глобально (если ещё не установлен):

  ```bash
  npm install -g @angular/cli

## 🚀 Запуск проекта
1. Клонировать репозиторий:

```bash
git clone <URL репозитория>
cd dashboard-angular
```

2. Установить зависимости:

```bash
npm install
```

3. Запустить сервер разработки:

```bash
ng serve
```

4. Открыть в браузере web-приложение по адресу http://localhost:4200
